## 기술 스택

### Frontend

- **Framework**: React (with Vite or Create React App)
- **Language**: TypeScript
- **Styling**: Tailwind CSS or styled-components
- **State Management**: React Context API or Zustand
- **Routing**: React Router
- **Charts**: Chart.js or Recharts (통계 시각화용)
- **Animation**: Framer Motion (카드 생성 애니메이션용)

### Backend/Database

- **Database**: Supabase
  - PostgreSQL database
  - Authentication (선택적)
  - Row Level Security
  - Realtime subscriptions (실시간 통계 업데이트)

### Deployment

- **Hosting**: Vercel or Netlify
- **Domain**: (추후 결정)

## DB 스키마 (Supabase)

### surveys 테이블

```sql
- id (uuid, primary key)
- created_at (timestamp)
- user_id (text) - 42 인트라 ID 또는 익명
- mbti (text)
- mbti_reliability (int)
- dev_field (text)
- preferred_language (text)
- active_time (text)
- favorite_snack (text)
- work_style (text) - 협업 vs 개인
- planning_style (text) - 계획형 vs 즉흥형
```

### survey_stats 테이블 (집계용, 선택적)

```sql
- id (uuid, primary key)
- total_responses (int)
- updated_at (timestamp)
- field_stats (jsonb) - 각 분야별 통계
```

## 주요 기능

### 사용자 기능

1. **설문 조사 페이지**

   - 크리스마스 테마 UI
   - 단계별 질문 (스텝 바이 스텝)
   - 진행률 표시

2. **통계 대시보드**
   - 실시간 투표 결과
   - 각 분야별 퍼센티지 차트
   - 재미있는 인사이트 제공

### 관리자 기능

- 관리자 URL을 통한 설문 관리
- 통계 데이터 확인
- 응답 데이터 내보내기 (CSV)

## 구현 단계

1. **Phase 1: 프로젝트 셋업**

   - React + Vite 프로젝트 생성
   - Supabase 프로젝트 생성 및 연결
   - 기본 라우팅 구조 설정

2. **Phase 2: 설문 조사 UI**

   - 설문 질문 컴포넌트 제작
   - 크리스마스 테마 디자인 적용
   - 폼 유효성 검사

3. **Phase 3: Supabase 연동**

   - DB 스키마 생성
   - 설문 응답 저장 로직
   - 실시간 통계 조회

4. **Phase 4: 결과 카드 생성**

   - Canvas API 또는 html2canvas 사용
   - 개발자 카드 템플릿 디자인
   - 이미지 다운로드 기능

5. **Phase 5: 통계 대시보드**

   - 차트 컴포넌트 구현
   - 실시간 데이터 업데이트
   - 인사이트 텍스트 생성

6. **Phase 6: 배포**
   - Vercel/Netlify 배포
   - 도메인 연결 (선택적)
   - 성능 최적화
